### 1. cd food-recognition-api
### 2. python -m venv venv
### 3. venv\Scripts\activate
### 4. pip install -r requirements.txt
### 5. Start the API server:
- uvicorn main:app --reload
### 6. Open the API in your browser:
- http://127.0.0.1:8000/docs
- Find POST /predict
- Click Try it out
- Upload a food image
- Click Execute
